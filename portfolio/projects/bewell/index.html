Bewell

RoleUX & Motion
Year2025
ClientBewell
ToolsFigma · Houdini · AE
Intro

A wellness app with atmosphere, not just icons.

Bewell is a meditation and breathing app looking to differentiate itself in a crowded market. The design brief asked for something more atmospheric than typical wellness UI — a visual language that felt like weather, not widgets.

My role spanned UX flow design and the generative particle system used throughout the onboarding and breathing-exercise screens.

Moodboard

Weather, not widgets

The visual references came from long-exposure photography, volumetric light studies, and the slow, inevitable motion of fog.

Story

Particles that breathe with the user

The centerpiece of the visual system was a particle field that responded to the user's breathing exercise — expanding on the inhale, contracting on the exhale, drifting laterally between phases. Building this in Houdini first gave us room to experiment with behavior, then export a lightweight version that could run in real time on a mobile device.

The UX flow itself was stripped back deliberately. Onboarding runs in three screens. The main surface is almost empty — just the particle field and a single breathing prompt. Every secondary action lives one tap deep.

The hardest part of the project was resisting the urge to add more. Each new element we tested diluted the atmosphere, so the final version is closer to the first sketch than any of the intermediate iterations.

Result

Quiet, responsive, atmospheric

The onboarding, breathing exercise, and main dashboard were delivered as a high-fidelity prototype with a custom particle system spec for the engineering team.
